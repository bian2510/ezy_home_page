# PRDs

Product Requirements Documents, one per feature or bounded context.

**Naming convention:** numbered `PRD-NN-{context}.md` (e.g., PRD-04-compliance-execution.md)
**Example filename:** `PRD-01-iam.md`

See also: [DOMAIN.md](../../DOMAIN.md), [PROJECT.md](../../PROJECT.md) for business and project context.
